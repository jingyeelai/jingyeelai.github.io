Top 100 Hotels Worldwide

June 1, 2025

The “Top 100 Hotels Worldwide” is a comprehensive data analysis and visualization project aimed at identifying the highest-rated hotels across continents based on online reviews, price tiers, and geographic locations.

  • Roles: Data Analyst, Visualization Designer
  • Year: 2025
  • Tools Used: Python, pandas, Tableau, MySQL
  • GitHub: evielai/top-100-hotels
  • Report: Download PDF

Project Process and Methodology

  1. Data Collection: Scraped hotel listings and reviews from Booking.com, TripAdvisor, and Google Maps.
  2. Cleaning and Standardization: Removed duplicates, normalized rating systems, and translated multilingual content.
  3. Analysis: Performed cluster analysis to segment hotels and identified standout attributes across regions.
  4. Visualization: Created dashboards to present findings, highlight customer satisfaction, and pricing trends.

Data Source

  • Booking.com: Hotel reviews and metadata
  • TripAdvisor: Customer feedback and ratings
  • Google Maps: Geo-location and travel accessibility

Challenges

  • Inconsistent Formats: Different rating scales and review languages.
  • Duplicate Listings: Merged data across platforms required fuzzy matching.

Solutions

  • Regex-based Cleaning: Standardized fields such as location and review format.
  • Fuzzy Matching: Applied fuzzy logic to combine similar listings.

Technologies Used

  • Python (data processing and scraping)
  • pandas, scikit-learn
  • Tableau (visualization)
  • MySQL (data storage)
